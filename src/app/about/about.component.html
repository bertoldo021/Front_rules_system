Sobre o Projeto

O sistema coleta as informações fornecidas pelo usuário na interface "Obter Recomendação". Em seguida, envia esses dados para o backend, que é implementado em Python e Clips, onde um sistema baseado em regras processa as informações. Como resultado desse processamento, o sistema retorna uma tabela contendo as recomendações relevantes, que são exibidas ao lado da interface para que o usuário as acesse facilmente. Esse fluxo de trabalho permite que o usuário obtenha orientações personalizadas com base em suas entradas, simplificando o processo de tomada de decisões.
